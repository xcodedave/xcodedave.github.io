Experiments

  • Forlorn

    An ASCII 3D game rendered entirely in a terminal-style monospace grid. Walk through a desolate landscape with ambient desert sounds and rain.

  • 7,000,000,000

    A WebGPU space experience built in Rust/WASM. Navigate a vast, atmospheric scene with spatial audio.

  • SwiftVoxel

    A WebGPU voxel editor running in the browser. Place and remove blocks to sculpt 3D structures in real time.

  • Flight

    A Rust/WASM flight simulation. A minimal canvas experiment exploring movement and 3D geometry.

  • AVD ↔ Lottie Converter

    Convert Android Vector Drawable (AVD) animation files to Lottie JSON format, entirely in the browser via WASM.

  • Screech Chat

    A minimal real-time chat interface experiment.

  • Device Code Phishing PoC

    A proof-of-concept demonstrating the OAuth device code flow as a phishing vector, for security research and awareness.

  • Birds (Experiment No.1)

    An early Emscripten/WASM experiment — a flock of birds simulated in the browser.
